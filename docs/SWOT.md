# SWOT — Audiomancer

> Last updated: 2026-03-26

## Strengths
- Pure Python, zero heavy dependencies (numpy/scipy only)
- 14 modules, 15 scripts, full test coverage
- Direct integration with Akasha Portal pipeline

## Weaknesses
- No real-time preview (batch-only)
- No GUI — CLI scripts only

## Opportunities
- Progressive/loopable stems (compose.py — planned)
- Spatial audio expansion (already started)
- Reusable as standalone library (PyPI)

## Threats
- Scope creep toward DAW features
- Audio quality ceiling without DSP expertise
